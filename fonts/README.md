# Fonts

Custom fonts are not included in this repository. Download and place them in the correct folders before running.

Set `FONT_COUNTDOWN` in `config.py` to the `.ttf` path, or leave it as an empty string to use the system default.

## Example

[Bebas Neue](https://fonts.google.com/specimen/Bebas+Neue)

- File location: `fonts/Bebas_Neue/BebasNeue-Regular.ttf`
- In `config.py`: `FONT_COUNTDOWN: str = "fonts/Bebas_Neue/BebasNeue-Regular.ttf"`
